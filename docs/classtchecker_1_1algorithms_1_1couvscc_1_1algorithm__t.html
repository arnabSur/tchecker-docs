Detailed Description

template<class TS, class GRAPH>
class tchecker::algorithms::couvscc::algorithm_t< TS, GRAPH >

Couvreur's SCC-decomposition-based liveness algorithm.

Template Parameters
TS: type of transition system, should derive from tchecker::ts::ts_t
GRAPH: type of graph, should derive from tchecker::graph::reachability_graph_t, and nodes of type GRAPH::shared_node_t should derive from tchecker::algorithms::couvscc::node_t and have a method state_ptr() that yields a pointer to the corresponding state in TS
Note
Our implementation is based on the vairant of Couvreur's algorithm in: "Comparison of Algorithms for Checking Emptiness on Büchi Automata", Andreas Gaiser and Stefan Schwoon MEMICS 2009

We have implemented the iterative translation of the recursive algorithm in Gaiser&Schwoon's paper. The algorithm below works for generalised Büchi conditions.

procedure couvscc(A) count := 0 Roots := {} Active := {} Todo := {} for each initial state s0 of A if s0.dfsnum = 0 then couv_dfs(s0)

procedure push(s) count := count + 1 s.dfsnum := count s.current := true push(Roots, <s, labels(s)>) push(Active, <s>) push(Todo, <s, post(s)>)

procedure couv_dfs(s) push(s) while (Todo is not empty) do <n, succ> := top(Todo) if (succ is empty) then if (top(Roots) = <n, ?>) then close_scc(n) pop(Todo) else t := pick(succ) if (t.dfsnum = 0) then push(t) else if t.current then merge_scc(t)

procedure close_scc(n) pop(Roots) repeat u := pop(Active) u.current := false until u = n

procedure merge_scc(t) L := {} repeat <u, C> := pop(Roots) L := L + C if (L is accepting) then report cycle until u.dfsnum <= t.dfsnum push(Roots, <u, L>)

Check if a transition has an infinite run that satisfies a given set of labels and build the corresponding graph.

Parameters
ts: a transition system
graph: a graph
labels: accepting labels
Postcondition
graph is built from a traversal of ts starting from its initial states, until a cycle that satisfies labels is reached (if any). A node is created for each reached state in ts, and an edge is created for each transition in ts.
Returns
statistics on the run
Note
if labels is empty, graph is the full state-space of ts
