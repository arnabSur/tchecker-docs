Detailed Description

template<class NODE_PTR, class EDGE_PTR>
class tchecker::graph::directed::graph_t< NODE_PTR, EDGE_PTR >

Directed graph.

Template Parameters
NODE_PTR: type of pointers to node, *NODE_PTR should derive from tchecker::graph::directed::node_t
EDGE_PTR: type of pointers to edge, *EDGE_PTR should derive from tchecker::graph::directed::edge_t
Note
This graph does not allocate and does not store nodes and edges. It only links nodes and edges. Edges are stored in nodes.
