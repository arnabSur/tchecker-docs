Detailed Description

template<class NODE_SPTR, class NODE_SPTR_HASH, class NODE_SPTR_EQUAL>
class tchecker::graph::find::graph_t< NODE_SPTR, NODE_SPTR_HASH, NODE_SPTR_EQUAL >

Graph with node finding.

Template Parameters
NODE_SPTR: type of shared pointer to node, the pointed node should inherit from tchecker::hashtable_object_t
NODE_SPTR_HASH: hash function on nodes pointed by NODE_SPTR, should return the same hash code for nodes which are equal w.r.t. EQUAL
NODE_SPTR_EQUAL: equality function on nodes pointed by NODE_SPTR
Note
this graph implementation stores nodes and answers find queries. It does not store edges
each node has a unique instance in this graph w.r.t. NODE_SPTR_EQUAL
