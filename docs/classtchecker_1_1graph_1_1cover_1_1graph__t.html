Detailed Description

template<class NODE_SPTR, class NODE_SPTR_HASH, class NODE_SPTR_LE>
class tchecker::graph::cover::graph_t< NODE_SPTR, NODE_SPTR_HASH, NODE_SPTR_LE >

Graph with node covering.

Template Parameters
NODE_SPTR: type of shared pointer to node
NODE_SPTR_HASH: type of hash function on node pointers. Should be callable with a NODE_SPTR argument and return a hash code for the pointed node
NODE_SPTR_LE: less-than-or-equal predicate on nodes. Should be callable with two NODE_SPTR argument and return true if the first one is smaller than the second one, and false otherwise. Usually, two nodes that are comparable w.r.t. NODE_SPTR_LE should have the same hash code returned by NODE_SPTR_HASH
Note
This graph allows to check if there is a node in the graph that covers some given node. Nodes are compared using NODE_SPTR_LE. Only the nodes with the same hash value w.r.t. NODE_SPTR_HASH are compared

Check if a node is covered in the graph.

Parameters
n: a node
covering_node: a node
Postcondition
covering_node is such that NODE_SPTR_LE(n, covering_node) is true if such node exists in the graph, nullptr otherwise
Returns
true if if a covering node has been found for n, false otherwise
Note
Only the nodes which have the same hash value than n w.r.t. NODE_SPTR_HASH will be considered as potential covering nodes
this function ensures that n is never covered by itself (i.e. it returns false if n is the only node in this graph for which NODE_SPTR_LE is true)
