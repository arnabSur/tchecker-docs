Detailed Description

template<class W>
class tchecker::waiting::fast_remove_waiting_t< W >

Waiting container that simulates fast removing of elements anywhere in the container.

Template Parameters
W: type of waiting container, should implement tchecker::waiting::waiting_t. The type of elements W::element_t should be a pointer to a type deriving from tchecker::waiting::element_t
