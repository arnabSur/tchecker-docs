Detailed Description

template<class NODE, class EDGE>
class tchecker::graph::finite_path_t< NODE, EDGE >

Finite path.

Template Parameters
NODE: type of nodes
EDGE: type of edges
Note
this path allocates nodes of type tchecker::graph::reachability::node_t<NODE, EDGE> and edges of type tchecker::graph::reachability::edge_t<NODE, EDGE>
the path is created empty. A first node can then added to the path. Then, the path can be extended from the front or from the back adding an edge and a node
