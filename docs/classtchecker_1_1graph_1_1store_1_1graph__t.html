Detailed Description

template<class NODE_PTR>
class tchecker::graph::store::graph_t< NODE_PTR >

Graph as an iterable collection of nodes.

Template Parameters
NODE_PTR: type of pointer to node
Note
this graph implementation stores nodes and allows iteration over the set of nodes. It does not store edges
