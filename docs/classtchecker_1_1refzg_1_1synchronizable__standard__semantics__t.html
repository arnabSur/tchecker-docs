Detailed Description

Standard semantics: each transition in the zone graph consists of a delay (if allowed) followed by a transition from the timed automaton, extended with synchronizable check on zones: only zones that contain a synchronized valuation are considered valid.

Compute initial zone with reference clocks.

Parameters
rdbm: a DBM
r: reference clocks for rdbm
delay_allowed: set of reference clocks allowed to delay
invariant: invariant
spread: reference clocks spread
Precondition
rdbm is not nullptr (checked by assertion). rdbm is a r.size()*r.size() array of difference bounds. rdbm is tight and consistent. the size of delay_allowed is the number of reference clocks in r (checked by assertion)
Postcondition
rdbm is the zone that only containts the zero valuation
Returns
tchecker::STATE_OK if the resulting DBM is not empty. Otherwise, tchecker::STATE_CLOCKS_EMPTY_SPREAD if the zero valuation is not spread bounded, tchecker::STATE_CLOCKS_SRC_INVARIANT_VIOLATED if the zero valuation does not satisfy invariant, tchecker::STATE_ZONE_EMPTY_SYNC if the resulting DBM does not contain any synchronized valuation.
Note
set spread to tchecker::refdbm::UNBOUNDED_SPREAD for unbounded spread

Compute next zone with reference clocks.

Parameters
rdbm: a DBM
r: reference clocks for dbm
src_delay_allowed: set of reference clocks allowed to delay in source state
src_invariant: invariant in source state
sync_ref_clocks: set of reference clocks to synchronize
guard: transition guard
clkreset: transition reset
tgt_delay_allowed: set of reference clocks allowed to delay in target state
tgt_invariant: invariant in target state
spread: reference clocks spread
Precondition
rdbm is not nullptr (checked by assertion). rdbm is a r.size()*r.size() array of difference bounds. rdbm is not tight and consistent. the size of src_delay_allowed is the number of reference clocks in r (checked by assertion). The size of sync_ref_clocks is the number of reference clocks in r (checked by assertion).) The size of tgt_delay_allowed is the number of reference clocks in r (checked by assertion).
Postcondition
rdbm has been delayed (only reference clocks which are allowed in src_delay_allowed), then the spread in rdbm has been bounded to spread, then intersected with src_invariant, then all reference clocks in sync_ref_clocks have been synchronized in rdbm, then rdbm has been intersected with guard, then dbm has been reset w.r.t clkreset, then intersected with tgt_invariant
Returns
tchecker::STATE_OK if the resulting DBM is not empty. Otherwise, tchecker::STATE_CLOCKS_SRC_INVARIANT_VIOLATED if intersection with src_invariant result in an empty zone, tchecker::STATE_CLOCKS_EMPTY_SPREAD if bounding the spread result in the empty zone, tchecker::STATE_CLOCKS_SYNC_EMPTY if synchronization of reference clocks in sync_ref_clocks yield an empty zone, tchecker::STATE_CLOCKS_GUARD_VIOLATED if intersection with guard result in an empty zone, tchecker::STATE_EMPTY_SYNC if synchronization of reference clocks result in an empty zone, tchecker::STATE_CLOCKS_TGT_INVARIANT_VIOLATED if intersection with tgt_invariant result in an empty zone, tchecker::STATE_ZONE_EMPTY_SYNC if the resulting zone does not contain any synchronized valuation
Note
set spread to tchecker::refdbm::UNBOUNDED_SPREAD for unbounded spread
