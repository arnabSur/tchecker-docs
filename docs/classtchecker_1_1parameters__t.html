Declare a bounded parameter.

Parameters
name: variable name
size: variabel size (array)
min: minimal value
max: maximal value
initial: initial value
Precondition
'name' is not a declared variable, size > 0, and 0 <= min <= max
Postcondition
A bounded parameter with base name 'name', with size 'size', with minimal value 'min', wirh maximal value 'max', and with initial value 'initial' has been declared
Exceptions
std::invalid_argument: if the precondition is violated
