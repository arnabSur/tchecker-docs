Detailed Description

template<class STATE>
class tchecker::pzg::details::state_pool_allocator_t< STATE >

Pool allocator for states of parametric zone graphs that can be extended to allocate more complex states.

Template Parameters
STATE: type of states, should inherit from tchecker::pzg::state_t and should be a tchecker::make_shared object
