Detailed Description

template<class STATE, class CONST_STATE, class TRANSITION, class CONST_TRANSITION, class INITIAL_RANGE, class OUTGOING_EDGES_RANGE, class INITIAL_VALUE = typename std::iterator_traits<typename INITIAL_RANGE::begin_iterator_t>::reference_type const, class OUTGOING_EDGES_VALUE = typename std::iterator_traits<typename OUTGOING_EDGES_RANGE::begin_iterator_t>::reference_type const>
class tchecker::ts::full_ts_t< STATE, CONST_STATE, TRANSITION, CONST_TRANSITION, INITIAL_RANGE, OUTGOING_EDGES_RANGE, INITIAL_VALUE, OUTGOING_EDGES_VALUE >

Transition system with full access to initial/outgoing edges.

Template Parameters
STATE: type of state
CONST_STATE: type of const state
TRANSITION: type of transition
CONST_TRANSITION: type of const transition
INITIAL_RANGE: type of range of initial states, should be tchecker::make_range_t<...>
OUTGOING_EDGES_RANGE: type of range of outgoing edges, should be tchecker::make_range_t<...>
INITIAL_VALUE: type of value in INITIAL_RANGE
OUTGOING_EDGES_VALUE: type of value in OUTGOING_EDGES_RANGE
