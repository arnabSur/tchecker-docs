Detailed Description

template<class SPTR, class HASH, class EQUAL>
class tchecker::hashtable_t< SPTR, HASH, EQUAL >

Hashtable.

Template Parameters
SPTR: type of pointer to stored objects. Must be a shared pointer, e.g. tchecker::intrusive_shared_ptr_t<...> or std::shared_ptr<...>
HASH: hash function over shared pointers of type SPTR
EQUAL: equality predicate over shared pointers of type SPTR
Note
stored objects should derive from tchecker::hashtable_object_t
