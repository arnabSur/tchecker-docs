Detailed Description

template<class SPTR, class HASH, class EQUAL>
class tchecker::cache_t< SPTR, HASH, EQUAL >

Cache of shared objects with collection of unused objects.

Template Parameters
SPTR: type of pointer to stored objects. Must be a shared pointer tchecker::intrusive_shared_ptr_t<...> to an object that derives from tchecker::cached_object_t
HASH: hash function over shared pointers of type SPTR, must be default constructible
EQUAL: equality predicate over shared pointers of type SPTR, must be default constructible
Note
stored objects should derive from tchecker::cached_object_t
