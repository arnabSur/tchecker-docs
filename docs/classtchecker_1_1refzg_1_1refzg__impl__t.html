Detailed Description

Low-level transition system implementation of zone graph with reference clocks of a timed automaton.

r: reference clock variables
semantics: a semantics over zones with reference clocks
spread: spread bound over reference clocks
block_size: number of objects allocated in a block
table_size: size of hash tables
Note
all states and transitions are pool allocated and deallocated automatically
set spread to tchecker::refdbm::UNBOUNDED_SPREAD for unbounded spread
