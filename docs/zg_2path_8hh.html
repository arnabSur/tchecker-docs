Compute a finite run in a zone graph following a specified sequence of tuples of edges.

Template Parameters
ZG: type of zone graph
VEDGE_RANGE: type of range of tuple of edges, shall dereference to tchecker::const_vedge_sptr_t
Parameters
zg: zone graph
initial_vloc: tuple of initial locations
seq: sequence of tuple of edges as a range
Precondition
initial_vloc is a tuple of initial locations in zg
seq is a feasible sequence in zg from initial_vloc
Returns
a finite path in refzg, starting from the initial state with tuple of locations initial_vloc, and that follows the sequence seq if possible. If the path is not empty, its first node has flag initial set to true
Exceptions
std::invalid_argument: if there is not initial state in refzg with tuple of locations initial_vloc or if seq is not feasible from the initial state
Note
the returned path keeps a shared pointer on zg
