Detailed Description

template<class SPTR, class HASH>
class tchecker::collision_table_t< SPTR, HASH >

Container of shared objects with hashing and collision lists, and fast remove.

Template Parameters
SPTR: type of pointer to stored objects. Must be a shared pointer, e.g. tchecker::intrusive_shared_ptr_t<...> or std::shared_ptr<...>
HASH: hash function over shared pointers of type SPTR
Note
stored objects should derive from tchecker::collision_table_object_t
collision tables do not check for object equality: objects with same hash value are simply stored in the same collision list.
